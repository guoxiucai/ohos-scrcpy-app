# 变更日志

所有显著变更将记录在此文件中。格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/)。

## [1.0.0] - 2026-05-16

### Added

- 实时投屏：H.264 硬件编码 + 平台原生硬件解码
- 触控注入：鼠标操控设备触摸屏
- 应用管理：HAP 安装 / 卸载
- 音量 / 亮度远程控制
- 功能键模拟：返回、主页、最近任务
- 内嵌 hdc shell 模拟终端
- 运行时动态参数调整（分辨率、码率、帧率）
- macOS 客户端（VideoToolbox 硬解）
- Windows 客户端（MediaFoundation 硬解）
- macOS 打包脚本（签名 + 公证）
- Windows 打包脚本（Inno Setup）
